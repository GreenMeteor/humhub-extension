# HumHub Installer

## Description

The HumHub Installer extension for Plesk enables easy installation and configuration of HumHub, a flexible open-source social network kit and framework. With this extension, you can quickly deploy HumHub instances on your Plesk server, making it simple to create social networking platforms for your users or clients.

## Features

- One-click installation of HumHub on your Plesk server.
- Streamlined configuration process for setting up HumHub instances.
- Automated setup of databases, permissions, and other requirements.
- Seamless integration with Plesk Obsidian's extension framework.

## Requirements

- Plesk Obsidian or later.
- Compatible with HumHub version `1.16.0-beta.1`.
